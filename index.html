Bot It

Bot It is a bot built on Python, Django, and Selenium made to automate the process of purchasing highly demanded, limited release Adidas sneakers. As a wannabe sneakerhead, it was almost impossible to buy sneakers that I wanted manually because of how quickly they were being sold out. As a result, I decided to create a bot that would find the shoe size, solve the Captcha, and add the desired shoe to cart, significantly speeding up the purchase time and increasing the probability of actually getting the shoes.
I presented the design to judges from over 10 companies at the IEEE Project Showcase and won the best overall underclassman project award. I also pitched the idea to The Garage, a startup incubator at Northwestern, and was accepted to be a startup resident team for the spring quarter, where I worked on monetizing the software and usage of the bot.

Weight Watcher

Weight Watcher is a program I made with 2 other team members as part of our Design Thinking and Communication course at Northwestern. Our client was a physical therapist from the Rehabilitation Institute of Chicago (RIC). The RIC uses a number of different body weight support treadmills, but the problem was that the harnesses on the treadmills had no actual measure of the weight being offloaded from a patient. As a result, we were given 10 weeks to engineer a mechanism to accurately measure and display the weight being offloaded by the harness.
Using two load cells, and a wifi-enabled microcontroller, we were able to read the voltages across each load cell when it was being pulled by the harness, and then convert the voltage drop to an actual weight in pounds. This data was then transferred to a live and locally hosted web page for the physical therapists to view.
We presented our design at Northwestern's annual Design Expo, and won the Communications Award for best presentation and documentation of design out of over 60 other engineering teams.

Power Flow Neural Net

As part of my internship at Argonne this summer, I've been working on building and implementing a number of different machine learning algorithms. One of those algorithms I've developed is a multilayer perception neural network (for regression) using Python and Tensorflow. This neural network takes in data in the form of a CSV, splits the data into training and testing sets, and then using forward and back propagation, will iterate through a specified number of epochs, and learn the data, minimizing error on each epoch using stochastic gradient descent. There are also functions to test accuracy on the testing set, and use the learned model to make predictions.
Using the IEEE 14 bus test case, I generated a dataset of 3500 samples, and tried to predict the difference between the optimal cost of the AC model vs the DC model. Using the neural net, I was able to train a model to predict the cost difference with only 0.8 squared error.
